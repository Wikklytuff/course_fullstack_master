- 浏览器的前进后退功能设计
    1. 不断地访问网页，并且入栈留下历史记录以便后退
        而到了栈底不可进行后退操作，栈空的时候 FILO
    2. 使用两个栈XY，依次将abc页面放入X
        后退操作就是将cba放入Y
        前进操作就是将abc放回X
    3. 后退时为了可以前进，X出栈，Y入栈
        前进时，Y出栈，X入栈
    4. 当Y有数据的时候，X入栈
        X正常入栈，Y清空

- 34+139+44-12/3
    2个栈 XY
    X用来存数，Y用来存运算符（优先级）
    Y出栈 X栈应该弹出两次 因为双目运算符一定是两个数
    若新入的运算符比栈顶优先级高， 就将当前运算符压入栈；若优先级低或相同，就从Y栈取栈顶运算符，从X栈取2个数，进行计算后压入X栈，继续。
